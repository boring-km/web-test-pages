왜 필요한가

현재 앱은 단독 학습에 강점이 있지만, 다시 돌아오게 만드는 감정적 연결과 관계 기반 동기가 약합니다. 이번 기획은 학습과 소셜을 한 흐름으로 묶어 리텐션을 올리는 데 초점을 둡니다.

  1. 내 캐릭터가 말을 건다
  2. 내 소개와 정체성을 설정한다
  3. 친구에게 공유하고 연결한다
  4. 레벨/칭호로 성장 체감을 얻는다
  5. 다시 앱에 들어올 이유가 생긴다

핵심 기능 구성

캐릭터 대사 말풍선

홈 진입 시 자동 대사 노출, 탭 시 새로운 대사로 전환. 하단 번역 표시를 통해 학습 흥미와 이해도를 함께 올립니다.

자기소개 한 줄

짧은 소개 문구로 캐릭터에 개성을 부여합니다. 친구가 보는 프로필에도 동일하게 반영되어 정체성 표현이 강화됩니다.

친구 추가/목록

친구 코드와 공유 링크 기반으로 연결을 단순화합니다. 요청/수락 구조를 통해 안전하게 관계를 형성합니다.

친구 프로필 보기

캐릭터, 닉네임, 소개, 레벨 정보를 읽기 전용으로 보여주어 비교와 동기부여 경험을 만듭니다.

MZ력 성장 체계

활동별 경험치, 레벨, 칭호를 통해 학습 행동이 눈에 보이는 보상으로 연결됩니다. 레벨업 연출로 성취감을 강화합니다.

친구 탭 중심 허브

요청 확인, 친구 추가, 목록 탐색, 프로필 열람까지 한 화면 흐름으로 구성해 소셜 진입 장벽을 낮춥니다.

시각 자료로 보는 기능

텍스트만 읽기 어려운 부분을 이미지와 아이콘으로 빠르게 이해할 수 있게 구성했습니다.

홈 말풍선 경험

앱에 들어오자마자 캐릭터가 말을 걸어 학습 시작의 심리적 장벽을 낮춥니다.

자기소개 편집

짧은 한 줄 소개로 캐릭터 정체성을 표현하고 친구 화면에서도 동일하게 노출합니다.

친구 연결 흐름

코드 공유와 요청 수락 과정을 통해 안전하고 명확하게 친구 관계를 형성합니다.

성장 체감

경험치, 레벨, 칭호가 함께 보여서 단기 보상과 장기 목표를 동시에 제공합니다.

상세 기능 설명

아래는 실제 사용자가 체감하게 될 동작 기준으로 정리한 상세 시나리오입니다. 기술 구현 방식은 제외하고, 화면에서 보이는 경험만 담았습니다.

1) 캐릭터 대사 + 번역 인터랙션

홈 화면 진입 순간 캐릭터가 말을 걸어 "앱이 살아있다"는 느낌을 주고, 탭 행동으로 다음 학습 자극을 이어갑니다.

  • 진입 시 대사 1개가 자동 노출되고, 캐릭터 탭 시 새 대사로 교체됩니다.
  • "탭해서 번역" 안내를 제공해 초보 학습자도 부담 없이 뜻을 확인할 수 있습니다.
  • 번역은 말풍선 아래 펼쳐져 원문과 의미를 함께 비교하도록 유도합니다.

2) 자기소개 한 줄 프로필

학습자마다 성향과 분위기를 표현할 수 있도록 짧고 간단한 자기소개를 중심 요소로 둡니다.

  • 짧은 길이 제한으로 작성 부담은 낮추고, 읽기 속도는 높입니다.
  • 홈과 친구 프로필 모두 동일하게 보여 일관된 자기 표현을 유지합니다.
  • 편집 진입이 쉬워, 사용자 상태 변화에 맞춰 자주 업데이트하도록 유도합니다.

3) 친구 추가와 요청 확인 흐름

친구 연결은 빠르고 단순해야 하므로 코드 입력/공유 링크 중심으로 설계하며, 즉시 친구가 되지 않고 요청-수락 단계를 둡니다.

  • 내 친구 코드를 항상 쉽게 확인하고 복사할 수 있어야 합니다.
  • 받은 요청은 배지와 안내 표시로 놓치지 않게 보여줍니다.
  • 요청 수락 후 친구 목록에 반영되어 "연결 완료"를 명확히 체감합니다.

4) 친구 프로필 열람 경험

친구 프로필은 복잡한 소셜 피드 대신, 캐릭터/소개/레벨 중심의 가벼운 비교 경험을 제공합니다.

  • 캐릭터 비주얼, 닉네임, 칭호, 소개 문구를 한 화면에서 직관적으로 확인합니다.
  • MZ력 레벨과 진행 상태를 보여 성장 정도를 서로 비교할 수 있습니다.
  • 관계 정리가 필요할 때는 삭제 동선을 명확히 제공해 불편을 줄입니다.

5) MZ력 보상 체계

퀴즈/출석/친구 활동처럼 이미 존재하는 행동에 보상 레이어를 덧씌워, 학습 루틴이 성장 루틴으로 인식되게 만듭니다.

  • 행동 직후 경험치 획득 피드백으로 "무엇을 잘했는지"를 즉시 전달합니다.
  • 경험치와 레벨을 함께 보여 단기 성취와 장기 목표를 동시에 제공합니다.
  • 레벨별 칭호 변화로 성장의 의미를 숫자 이상으로 느끼게 합니다.

사용자 경험 원칙

즉각적 피드백

경험치 획득과 레벨업은 즉시 피드백으로 노출해 행동 보상을 명확하게 전달합니다.

부담 없는 상호작용

메시지 기능 없이도 친구 연결과 상태 확인만으로 가벼운 소셜 재미를 제공합니다.

학습과 재미의 결합

대사, 번역, 성장 요소를 분리하지 않고 하나의 홈 경험 안에서 자연스럽게 연결합니다.

확장 가능한 구조

향후 업적 칭호, 상호작용 이벤트, 대사 확장 등으로 무리 없이 확장할 수 있도록 기획 단계에서 여지를 둡니다.

번역 토글 UX 세션 단위 알림 노출 친구 최근순 탐색 삭제 시 관계 정리 명확화

도입 순서 (기획 관점)

기술 구현 순서가 아니라, 사용자 가치가 빨리 보이는 흐름을 우선으로 정리한 단계입니다.

1단계 - 홈에서 변화 체감

말풍선 대사, 소개 문구, 레벨/칭호 표시로 캐릭터 중심 경험을 먼저 강화합니다.

2단계 - 친구 연결 진입

코드/링크 기반 추가와 요청 확인 흐름을 붙여 소셜 루프를 시작합니다.

3단계 - 비교와 성장 고정

친구 프로필 열람과 경험치 피드백을 통해 장기 동기를 만들고 재방문을 유도합니다.

주의: 초기에 기능을 과도하게 늘리기보다, "친구 연결 + 성장 체감"이 반복되는 핵심 루프 완성도를 우선합니다.

MVP 범위와 후속 확장

MVP 포함

대사/번역, 소개 편집, 친구 요청/수락/삭제, 친구 프로필 열람, 경험치/레벨/칭호 피드백.

후속 확장

업적형 칭호, 친구 상호작용 미니 기능, 더 풍부한 대사 소스, 고도화된 알림/안전 기능.
